# Видео Reels — дюжина дня

Каждый день в **08:00 по Москве** на странице появляется свежая дюжина рилсов:
https://trainingsw75-cloud.github.io/reels/

Как пополнять копилку: увидел рилс в Facebook → «Поделиться» →
«Скопировать ссылку» → прислать ссылку Telegram-боту (секрет `REELS_BOT_TOKEN`).
Робот забирает ссылки каждые 2 часа (`collector.py`), утром `build_reels.py`
собирает страницу: сперва ещё не выходившие ссылки, потом по кругу.

Видео не скачиваются и не перезаливаются — показываются официальным
встроенным плеером Facebook, права остаются у авторов.
